Для установки сервиса:
    создать виртуальное окружение питона
    установить в него нужные библиотеки командой 
        pip install -r requirements.txt
    затем создать базу данных с именем flask в СУБД Postgres, а также пользователя flask_user с паролем password
    затем запустить файл db_create.py
    по окончании работы скрипта выполнить команду flask run
