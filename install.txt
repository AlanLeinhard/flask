Для установки сервиса:
    клонировать репазиторий
    выполнить команду:
        docker-compose build
    после сборки образа выполнить команду:
        docker-compose up -d
    затем нужно открыть терминал запущенного образа и выполнить в нем команду:
        python db_create.py
    готово, портал успешно установлен и готов к использованию
